# forum

This is forum plugin for Livestreet CMS.

## IMPORTANT

**To use this plugin requires Livestreet CMS version 0.5.1 or later.**

## Simple Install

To install forum do this:

	Download the latest build
	Rename dir to `forum`
	Upload
	Activate plugin
